Tepl 5 -> Tepl 6

  • The tepl_info_bar_set_buttons_orientation() now takes a GtkInfoBar parameter, not a TeplInfoBar.

  • The tepl_info_bar_add_close_button() function has been renamed to tepl_info_bar_setup_close_button().

  • The tepl_info_bar_add_icon() function has been replaced by tepl_info_bar_set_icon_from_message_type().

  • A location parameter has been added to the tepl_info_bar_add_content_widget() function.

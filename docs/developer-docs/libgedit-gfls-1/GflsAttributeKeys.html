Description

GflsAttributeKeys is a small utility to build a list of GFile attribute keys. The result is a single string to pass as an argument to a function like g_file_query_info().

It becomes useful to do only a single query with a list of keys coming from different parts of the program. To take an example, Gfls might have its own list of keys to query, but it is desirable to ask the application for additional keys.

Configure and use gedit plugins

You can do more with gedit by using plugins. Several plugins are installed by default, but a large number of other plugins are available separately.

Many Linux distributions make a group of these plugins available as a gedit-plugins package. Install the gedit-plugins package to make these extra plugins available to gedit.

Default gedit Plugins

Document statistics — View document statistics.
File browser panel — Use the side panel to browse and open files.
Insert date/time — Insert the current date/time at the cursor position.
Modelines — Emacs, Kate and Vim-style modelines support for gedit.
Sort — Sort lines of text into alphabetical order.
Spell-check your document — Check your files for spelling errors.
Text Size — Increase or decrease text size.

Additional gedit Plugins

Additional gedit plugins — How to install third-party plugins.
Bookmarks — Record commonly-referenced locations with bookmarks.
Bracket Completion — Auto-complete closing brackets.
Character Map — Insert special characters just by clicking on them.
Code comment — Comment-out or uncomment a block of code.
Color Picker — Pick a color from a dialog and insert its hexadecimal representation.
Draw Spaces — Show white space between words or portions of code.
Embedded Terminal — Add a terminal console to the bottom panel.
Join/Split Lines — Join several lines of text or split long lines.
Multi Edit — Edit a document in multiple places at once.
Session Saver — Save and restore your working sessions
Word Completion — Autocomplete words or portions of code as you type.

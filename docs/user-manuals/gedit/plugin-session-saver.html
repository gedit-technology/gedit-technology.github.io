Session Saver

The Session Saver plugin allows you to remember which documents are currently open and open the same documents again at a later time.

Enable Session Saver Plugin

To enable the plugin:

  1. Press the menu button in the top-right corner of the window and select Preferences ▸ Plugins.

  2. Select Session Saver to enable the plugin.

Save a session

Save all your open files to be able to open them later.

To save a session:

  1. Press the menu button in the top-right corner of the window and select Tools ▸ Save Session….

Open a session

Open all the files from an already saved session.

To open a session:

  1. Press the menu button in the top-right corner of the window and select Tools.

  2. Select one of the saved sessions from the menu.

Manage Sessions

Open and delete your already saved sessions.

Manage sessions

  1. Press the menu button in the top-right corner of the window and select Tools ▸ Manage Saved Sessions…
